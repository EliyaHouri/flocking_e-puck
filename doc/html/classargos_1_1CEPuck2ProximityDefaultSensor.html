◆ CalculateReading()

Real argos::CEPuck2ProximityDefaultSensor::CalculateReading ( Real  f_distance)
virtual

Calculates the proximity reading when the closest occluding object is located as the given distance.

Parameters
f_distanceThe distance of the closest occluding object in meters
Returns
A value in the range [0:4095], where 0 means that the object is too far to be sensed, and 4095 means the object is so close that it saturates the sensor.
