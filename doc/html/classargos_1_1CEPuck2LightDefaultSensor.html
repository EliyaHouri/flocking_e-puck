◆ CalculateReading()

Real argos::CEPuck2LightDefaultSensor::CalculateReading ( Real  f_distance,
Real  f_intensity 
)
virtual

Calculates the light reading resulting from a light source at the given distance. Denoting the intensity with i and the distance x, this function calculates i = (I / x)^2. I is the reference intensity of the light, that is, the distance at which the light reading saturates. It is dependent on the light entity being considered.

Parameters
f_distanceThe distance of the considered light source.
f_intensityThe reference intensity of the considered light source.
Returns
A value in the range [0:4095], where 0 means that the light is too far to be sensed, and 4095 means the light is so close that it saturates the sensor.
